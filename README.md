# BillEase

## Project Overview

BillEase is a modern, user-friendly web application designed to simplify group bill splitting and expense management. The platform eliminates the awkwardness of money-related conversations by providing transparent, fair, and stress-free expense tracking for groups, roommates, trips, and events.

### Key Features

- **Smart Bill Splitting**: Split expenses evenly, by percentage, or custom amounts
- **Expense Tracker Dashboard**: Clear view of who paid, who owes, and outstanding balances
- **Groups & Events**: Create dedicated spaces for trips, roommates, subscriptions, or one-time plans
- **Automated Reminders**: Polite, timely reminders sent automatically to avoid chasing payments
- **Multi-Currency Support**: Handle bills in NGN, USD, GBP, EUR, and more
- **Automated Settlement**: Smart suggestions for the simplest way to settle up
- **Dark/Light Theme Toggle**: User preference for visual comfort
- **Responsive Design**: Optimized for mobile, tablet, and desktop experiences

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with responsive design using media queries
- **Icons**: Font Awesome 6
- **Fonts**: Google Fonts (Oswald, Roboto)
- **Animations**: CSS animations and JavaScript-powered effects
- **Storage**: LocalStorage for cart and theme preferences

## Installation & Setup

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side dependencies required - runs entirely in the browser

### Running the Application

1. **Clone or Download** the project files to your local machine

2. **Navigate** to the project directory:
   ```bash
   cd /path/to/BillEase
   ```
```

## Usage

### For Users

1. **Browse the Landing Page**: Explore BillEase features and pricing plans
2. **Select a Plan**: Choose from Free, Pro, or Business tiers
3. **Add to Cart**: Click "Select" on any pricing plan to add it to your cart
4. **Manage Cart**: View, modify quantities, or remove items from your cart
5. **Contact Support**: Use the contact form to reach out with questions
6. **Theme Toggle**: Switch between light and dark modes using the moon/sun icon

### Navigation Features

- **Responsive Navbar**: Collapses to hamburger menu on mobile devices
- **Smooth Scrolling**: Click navigation links for seamless section transitions
- **Sticky Navigation**: Navbar remains visible while scrolling
